# Personal Portfolio — Huiyi (Angela) Sun

Light-blue minimalist portfolio site for resume and GitHub project links.

**Live site:** https://huiyisun-angela.github.io

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Structure

- `index.html` — single-page layout
- `css/style.css` — light blue theme
- `js/main.js` — nav scroll & mobile menu
- `resume/` — downloadable PDF

## Deploy

Push to the `huiyisun-angela.github.io` repository on GitHub; Pages serves from `main` branch root.
